Данный проект представляет собой Telegram-бота с двумя реализациями: синхронной (sync_bot.py) и асинхронной (async_bot.py). Бот умеет отвечать на команды, присылать цитату дня (с сайта citaty.info), свежие новости с «Фонтанки» и «Хабра», а также текущую погоду для любого города (через OpenWeatherMap API).
Бот поддерживает следующие команды (в обеих версиях):

/quote – присылает случайную цитату дня с сайта citaty.info.

/headlines (только в асинхронной версии) – присылает последние 5 заголовков новостей с сайтов Фонтанка.ру и Habr.

/weather (только в асинхронной версии) – запрашивает у пользователя название города, затем присылает текущую погоду (температура, ощущается как, давление, влажность, ветер).

Любой другой текст – бот просто повторяет его (эхо-режим).

В синхронной версии доступны только /quote и эхо-режим.
